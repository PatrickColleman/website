Firewall

 

Er bestaan meerdere soorten firewalls. Op volgende pagina's worden deze kort besproken.

Een kleine opsomming:

  • packet filter router
    • stateful inspection firewall
  • application level gateway
  • circuit level gateway
  • hybriede firewall

 

  • bastion host

 

Het is nodig op te merken dat in de praktijk dikwijls een mix van verschillende soorten firewalls gebruikt wordt, zelfs op verschillende computers geimplementeerd om de nodige processingkracht te kunnen leveren. Men noemt dit hybriede firewalls: deze zijn een combinatie: eerst gaat het frame door een packet filter. Wanneer het toegelaten wordt gaat het nog door de application level gateway. Men kan het dan echter ook onmiddellijk doorlaten.
