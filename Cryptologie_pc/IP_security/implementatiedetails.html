IPSec

IPSec implementaties

Bovenstaande figuur toont enkele combinaties van SA's.

In deel (a) is de beveiliging geraliseerd doordat beide (eind)partners IPSec gebruiken.

In deel (b) is de beveiliging alleen gerealiseerd tussen gateways (firewall, router). De hosts gebruiken geen IPSec. Deze methode is een goede mogelijkheid voor een VPN.

In geval (c) is er end-to-end security. De gateway to gateway beveiliging zorgt voor authenticatie en/of beveiliging. De hosts introduceren zelf IPSec!

In deel (d) kan een remote host via het Internet toegang krijgen tot een computer binnen zijn bedrijf. Er is een tunnelmode tussen de gateway (firewall) en de remote host.
