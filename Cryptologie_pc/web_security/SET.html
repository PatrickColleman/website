SET

 

SET staat voor Secure Electronic Transaction. Het is bedoeld om creditcard transacties over het Internet te beveiligen.

 

SET levert drie services:
- een veilig communicatiekanaal tussen alle bij de betaling betrokken partijen
- vertrouwen door het gebruik van X.509
- privacy, omdat de informatie alleen voor de transactiepartijen beschikbaar is

 

De belangrijkste kenmerken van SET zijn:
- vertrouwelijkheid van informatie (encryptie met DES)
- data-integriteit: de betalingsinformatie bestaat uit bestelinformatie, persoonlijke gegevens en betaalinstructies. Deze informatie wordt onderweg niet veranderd: controle door RSA digitale handtekening, volgens SHA-1 hash code.
- authenticiteit van de kaarthouder: X.509 certificaten met RSA handtekening
- leveranciersauthenticiteit: de leverancier moet een relatie hebben met een bank: X.509 certificaten met RSA handtekening
