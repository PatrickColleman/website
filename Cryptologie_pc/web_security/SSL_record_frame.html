SSL record protocol frame

De header bestaat uit volgende velden:
- content type (8 bits): hogere laag protocol
- major version (8 bits): SSL primair versienummer, momenteel gelijk aan 3
- minor version (8 bits): SSL secundair versienummer, momenteel gelijk aan 0
- compressed length (16 bits): lengte van de klare tekst. Deze waarde is maximaal gelijk aan 214 (=16364) + 2048.
